Política de Privacidad

Última actualización: 8 de abril de 2026

📋 Resumen ejecutivo

Parte es una aplicación 100% offline de control de personal. No recopila, no transmite ni comparte ningún dato personal con terceros. Toda la información que ingresas permanece almacenada únicamente en tu dispositivo.

1. Información general

Esta Política de Privacidad describe cómo Elkin Vargas ("el Desarrollador") gestiona la información en relación con el uso de la aplicación móvil Parte (identificador: com.elkin.parte).

Al utilizar la aplicación aceptas los términos descritos en este documento.

2. Datos que NO recopilamos

La aplicación no recopila ningún tipo de información personal, incluyendo:

  • Nombre, correo electrónico o datos de identificación
  • Ubicación geográfica
  • Contactos, fotos, cámara o micrófono
  • Información del dispositivo
  • Datos de uso o analítica
  • Información financiera
  • Cookies o tecnologías de rastreo

No se requiere registro, inicio de sesión ni conexión a internet.

3. Información almacenada en tu dispositivo

Toda la información se guarda exclusivamente en tu dispositivo mediante una base de datos local (SQLite). Ningún dato se transmite fuera de tu dispositivo.

4. Exportación de datos a PDF

  • El archivo se genera localmente
  • El usuario decide si lo comparte
  • El desarrollador no tiene acceso a estos archivos

5. Servicios de terceros

La aplicación no utiliza servicios de terceros como analítica, publicidad o rastreo.

Las tiendas de aplicaciones pueden recopilar datos según sus propias políticas:

  • Google Play: https://policies.google.com/privacy
  • App Store: https://www.apple.com/legal/privacy

6. Permisos del sistema

  • Almacenamiento: necesario para generar archivos PDF
  • Compartir archivos: permite enviar el PDF a otras apps

No se solicitan permisos sensibles.

7. Privacidad de menores

La aplicación no está dirigida a menores de edad y no recopila datos de ningún usuario.

8. Retención y eliminación de datos

  • Puedes eliminar datos dentro de la app
  • Al desinstalar la app se eliminan todos los datos

9. Cambios en esta política

Nos reservamos el derecho de actualizar esta política. Los cambios serán publicados en este mismo enlace.
